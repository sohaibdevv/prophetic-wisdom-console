# Prophetic Wisdom Console (Python Syntax Lab)

A simple console application demonstrating how to organize and display categorized data using Python Dictionaries. It features words of wisdom from Moses (Musa), verses from the Quran regarding what Jesus (Isa) said, and profound Hadiths from Prophet Muhammad (SAW), including his powerful message on human equality.

## What this teaches:
1. **Dictionaries:** Mapping keys (Categories) to multiple values (Quotes).
2. **Input Handling:** Capturing user choices to navigate different data paths.
3. **Looping:** Using `for...in` to iterate clearly through arrays of text.
4. **String Formatting:** Using f-strings (`f"..."`) and newline characters (`\n`) to display text cleanly.

## How to Run
Ensure you have Python installed, open your terminal, and run:

python main.py
